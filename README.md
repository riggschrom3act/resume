# Joshua Riggs - Software Engineer Portfolio

This repository contains a professional, fully responsive portfolio website built with HTML, CSS, and JavaScript.  
It demonstrates technical proficiency, design discipline, and a business-oriented engineering mindset, targeted at HR professionals, recruiters, and senior leaders in technology.

---

## Overview

The portfolio is designed to highlight both technical capability and professional impact.  
It presents projects that merge software engineering, data analytics, and operational problem-solving, particularly within regulated or finance-driven industries.

Key objectives:
- Present technical competence in front-end and full-stack fundamentals.
- Communicate results in measurable, business-oriented language.
- Provide a clean, accessible user experience with modern design and strong performance.

---

## Features

**Technical Features**
- Interactive project filtering and keyword search.
- Dark and light theme toggle with persisted user preference.
- Typewriter-style animated introduction text.
- Canvas-based data visualization of skills and proficiencies.
- Click-to-copy contact information and resume download.
- Accessible contact form (email client integration).
- Responsive, mobile-first design.
- SEO metadata, Open Graph, Twitter Cards, and JSON-LD schema for discoverability.

**Design & Performance**
- Fully responsive layout with no dependencies or frameworks.
- Native JavaScript only (no React, Vue, or libraries required).
- Minimal file size and fast load times.
- WCAG 2.1 accessibility-conscious markup.
